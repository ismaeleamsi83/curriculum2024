App Astronomy - Explore the Wonders of the Universe

HTML5 SCSS Angular 17 API Nasa

Mi aplicación ofrece una ventana al universo, permitiendo a los usuarios explorar datos y recursos astronómicos proporcionados por la API de la NASA de manera interactiva y educativa.

He utilizado la librería RxJS para el manejo de observables, el modulo HttpClientModule para realizar las peticiones a la API de la Nasa mediante filtrado por fechas separado en un Servicio y la carga diferida mediante la directiva @defer.
